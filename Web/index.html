Powerful Functions

One of the standout features is the seamless integration with Rate My Professors. As you browse for classes, TigerScheduler automatically displays the professor's rating right next to the course details. This allows you to make more informed decisions without ever leaving the page.

Instructions

Step 1: Download & Unzip

Click the download button to get the zip file. Once downloaded, extract the zip file then move the extracted folder to a safe location on your computer.

Warning

Removing the folder will delete the extension. Please move the folder to a safe place where it won't be accidentally deleted.

Step 2: Enable Developer Mode

Open your Chrome browser, go to `chrome://extensions`, and turn on the "Developer mode" toggle in the top-right corner.

Step 3: Load the Extension

Click on the "Load unpacked" button that appears and select the folder where you extracted the folder in Step 1.

Step 4: Start Scheduling!

The Better TigerScheduler extension is now installed! Navigate to the course registration page and enjoy the enhanced features.
